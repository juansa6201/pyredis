# Trabajo recuperatorio de redis

## Requerimientos 

* Python: 3.6
* Redis: 5.0.7

## Correr el servidor

* Correr el siguiente comando para levantar los contendores de docker

```
docker-compose up -d
```

## Cliente

* Correr el siguiente comando para leer los mensajes que envia el servidor

```
python3 get.py
```
